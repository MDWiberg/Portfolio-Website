Find Your Hat Game

Description

The goal of this program is to allow a user to input a field size (n x m), and then traverse the field to find their hat (^) to win the game.

The user moves by entering up, u, left, right, down or their first letters respectively (case insensitive). A user wins the game by moving the starting path character (*) to the hat or loses by falling in a hole (O) or moving off the field.

This project's main focus using and running Javascript in a server enviroment rather than a client-side environment

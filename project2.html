Credit Card Validator

Description

The goal of this web app is to recieve an input from a user to check if the credit/debit card number they input is valid or not and to report which credit card company it is from (Amex, Discover, MasterCard and Visa). This project was created using HTML, CSS and JavaScript and focuses on integrating JavaScript to bring functionality to the HTML page.

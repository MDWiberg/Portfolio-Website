Credit Card Validator

Description

The goal of this web app is to recieve an input credit/debit card number from a user to check if the number they input is valid or not.

The credit card company (Amex, Discover, MasterCard, Visa or otherwise unknown) is reported regardless if the card number is valid or not.

This project's main focus was to practice integrating JavaScript to give functionality to an otherwise static HTML page.
